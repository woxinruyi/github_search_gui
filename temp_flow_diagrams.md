# Git Auto Push 项目流程图文档

## 一、业务流程图 (Business Process Flow)

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              用户启动脚本                                     │
│                       python git_auto_push.py                                │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         Step 1: 解析命令行参数                                │
│   -p/--path: 仓库路径    -m/--message: 提交信息    -b/--branch: 分支名        │
│   -f/--force: 强制推送   -t/--token: Token        -u/--username: 用户名      │
│   --save-token: 保存Token  --delete-token: 删除Token                         │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         Step 2: 验证 Git 仓库                                 │
│                      检查目录是否包含 .git 文件夹                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                          ┌───────────┴───────────┐
                          │                       │
                          ▼                       ▼
                    [仓库有效]              [仓库无效] ──────► 返回错误并退出
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 3: 检查远程仓库配置                                  │
│                  执行 git remote 检查 origin 是否存在                          │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                          ┌───────────┴───────────┐
                          │                       │
                          ▼                       ▼
                  [已配置 origin]         [未配置] ──────► 提示配置命令并退出
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 4: Token 认证检查                                   │
│              检查是否提供 Token 或已保存 Token 配置                             │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                          ┌───────────┴───────────┐
                          │                       │
                          ▼                       ▼
                   [有 Token]              [无 Token]
                          │                       │
                          ▼                       ▼
              [配置 Token 认证]     ┌─────────────────────────┐
                          │        │  **自动获取 Token 流程** │
                          │        │  1. 打开浏览器           │
                          │        │  2. 跳转 GitHub 登录页   │
                          │        │  3. 用户输入账号密码      │
                          │        │  4. 跳转 Token 创建页    │
                          │        │  5. 自动填写 Token 信息   │
                          │        │  6. 获取生成的 Token     │
                          │        │  7. 保存到本地配置       │
                          │        └─────────────────────────┘
                          │                       │
                          └───────────┬───────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 5: 检查工作区状态                                   │
│                    执行 git status --porcelain                               │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                          ┌───────────┴───────────┐
                          │                       │
                          ▼                       ▼
                    [有更改]              [无更改] ──────► 提示无更改并退出
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 6: 暂存所有更改                                     │
│                         执行 git add -A                                      │
│                   (自动排除 .git_token_config.json)                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 7: 提交更改                                         │
│                 执行 git commit -m "提交信息"                                 │
│             (若未指定提交信息，自动生成带时间戳的信息)                           │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 8: 检查远程分支并推送                                │
│          执行 git ls-remote 检查分支 → git push -u origin <branch>           │

                                      │
                          ┌───────────┴───────────┐
                          │                       │

                          ▼                       ▼

                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 9: 恢复远程 URL                                     │
│                   移除 URL 中的 Token（保护安全）                              │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                           完成并返回结果                                      │
```

## 三、Token 自动获取流程图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         检测到无可用 Token                                    │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      询问用户是否自动获取 Token                               │
│                   "是否打开浏览器自动获取 GitHub Token?"                      │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                          ┌───────────┴───────────┐
                          │                       │
                          ▼                       ▼
                       [是]                    [否] ──────► 提示手动配置方法
                          │
                          ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 1: 启动浏览器 (Playwright)                          │
│                      打开 GitHub 登录页面                                     │
│                      https://github.com/login                                │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 2: 等待用户登录                                     │

│                      Step 3: 跳转到 Token 创建页面                            │

                                      │

                                      │

                                      │

┌─────────────────────────────────────────────────────────────────────────────┐
│                      Step 6: 获取生成的 Token                                 │

                                      │
                                      ▼

│                      Step 7: 保存 Token 到本地                                │

                                      ▼

```

## 五、配置文件说明

### Token 配置文件 (.git_token_config.json)

```json
{
  "token": "ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx",
  "username": "your_github_username",
  "created_at": "2026-03-17 09:26:00"
}
```

**安全说明**：
- 此文件已加入 `.gitignore`，不会被提交到远程仓库
- Token 仅在推送时临时写入远程 URL，推送后自动移除
- 建议定期更新 Token，设置合理的过期时间
