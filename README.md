# GitHub 热门项目搜索工具
一款基于 **GitHub REST API** 的开源项目热度搜索工具，支持多语言切换、热度排序、结果导出，帮你快速找到对应领域最热门的优质开源项目~
## 📖 项目简介
本工具可以帮助开发者快速搜索 GitHub 上热门的开源项目：
- ✅ 支持自定义搜索关键词
- ✅ 支持按 Star 数量 / Fork 数量 / 更新时间排序
- ✅ 支持指定编程语言过滤
- ✅ 支持中英文双语切换，界面友好
- ✅ 双击项目即可在浏览器打开 GitHub 主页
- ✅ 搜索结果自动保存为 JSON 文件方便后续分析
- ✅ 原生 GUI 界面，操作简单
## 🚀 如何使用
### 1. 环境要求
- Python 3.6+
- `requests` 库（本项目已依赖，如果没有请执行 `pip install requests`）
### 2. 运行程序
```bash
python github_search_gui.py
```
### 3. 使用说明
1. 在搜索框输入你要查找的关键词，比如 `machine learning`、`web framework` 等
2. 选择排序方式（默认按 Star 数量降序排列）
3. 选择需要过滤的编程语言（默认全语言搜索）
4. 设置需要显示多少条结果（最多显示 100 条）
5. 点击「搜索」按钮即可
6. 搜索完成后，双击任意项目就可以在浏览器打开项目查看详情
7. 结果会自动保存到 `github_search_results.json` 文件
## 🎨 界面预览
- **布局清晰**：控件分布合理，不会挤压遮挡
- **响应式布局**：支持窗口缩放，自适应不同大小
- **状态栏**：实时显示当前搜索状态与结果统计
- **多语言支持**：一键切换中英文
