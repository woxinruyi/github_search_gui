# GitHub Toolkit GUI 设计文档

## 一、功能概述

**GitHub 工具箱** 整合了两大核心功能：
1. **🔍 项目搜索** - 搜索 GitHub 热门开源项目
2. **📤 项目提交** - 提交本地项目到 GitHub

## 五、功能模块

### 5.1 项目搜索模块

| 功能 | 说明 |
|------|------|
| 关键词搜索 | 支持任意关键词搜索 |
| 排序方式 | Star 数量 / Fork 数量 / 最近更新 |
| 语言筛选 | Python / JavaScript / Java / C++ 等 |
| 结果数量 | 10-100 条可调 |
| 双击打开 | 在浏览器中打开项目页面 |
| 克隆项目 | 将选中项目克隆到本地 |

### 5.2 项目提交模块

| 功能 | 说明 |
|------|------|
| 仓库选择 | 浏览选择本地 Git 仓库 |
| 远程配置 | 自动获取远程仓库地址 |
| 分支选择 | master / main / develop |
| 提交信息 | 自定义或自动生成 |
| Token 管理 | 获取 / 保存 / 删除 Token |
| 提交并推送 | 完整流程 |
| 仅提交 | 只在本地提交 |
| 仅推送 | 只推送到远程 |

### 5.3 通用功能

| 功能 | 说明 |
|------|------|
| 多语言支持 | 中文 / 英文 切换 |
| 状态栏 | 显示当前操作状态 |
| 日志输出 | 彩色日志显示 |

## 六、使用方法

### 6.1 启动应用

```bash
python github_toolkit_gui.py
```

### 6.2 搜索项目

1. 切换到 **🔍 项目搜索** 标签页
2. 输入搜索关键词（如 "python machine learning"）
3. 选择排序方式和编程语言
4. 点击 **搜索** 按钮
5. 双击结果行打开项目页面
6. 选中项目后点击 **克隆项目** 下载到本地

### 6.3 提交项目

1. 切换到 **📤 项目提交** 标签页
2. 点击 **浏览** 选择本地 Git 仓库
3. 如需认证，点击 **获取** 获取 Token
4. 输入提交信息（可选）
5. 点击 **🚀 提交并推送** 完成操作

## 七、技术实现

### 7.1 使用的库

| 库 | 用途 |
|------|------|
| tkinter | GUI 框架 |
| ttk.Notebook | 标签页组件 |
| requests | GitHub API 请求 |
| threading | 后台线程执行 |
| webbrowser | 打开浏览器 |
| git_auto_push | Git 操作模块 |

### 7.2 线程安全

- 所有网络请求和 Git 操作在后台线程执行
- 使用 `root.after()` 安全更新 GUI
- 避免界面卡顿
